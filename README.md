# Log Analyzer
This is to make your work eazy while analysing through log files.

## What all things it can do.
* Formats the given raw log file in the analysable file
* You can use Multiple filters to filter out the relevent logs

## How to use:
* Clone this Repo
* Add your Log file in the Repo folder and rename it as sample.log
* In your CLI run
```bash
python main.py
```
